# FlipReact Plus - 專業電子書閱讀平台

一個現代化、高質量的電子書閱讀平台,採用最新的 UI/UX 設計理念打造。基於 React、TypeScript、Tailwind CSS 和 Supabase 構建,提供媲美 FlipBuilder 的專業閱讀體驗。

## ✨ 設計亮點

### 視覺設計
本平台採用現代化的設計語言,融合玻璃擬態效果、流暢的漸層配色和精心設計的動畫過渡,為用戶帶來賞心悅目的視覺體驗。每個組件都經過精心打磨,確保在各種裝置上都能呈現最佳效果。

### 用戶體驗
我們專注於創造直覺且流暢的用戶體驗。從上傳電子書到閱讀內容,每個操作流程都經過優化,減少不必要的步驟,提供即時的視覺反饋。無論是桌面電腦、平板還是手機,用戶都能享受一致且優質的體驗。

### 功能特色
平台提供完整的電子書管理功能,包括智能的 PDF 轉換處理、直覺的拖放上傳、靈活的縮放控制、便利的書籤系統,以及專業的分享功能。所有功能都以用戶為中心,確保操作簡單且高效。

## 🎯 核心功能

### 管理後台
管理後台採用卡片式佈局,讓您能夠一目了然地查看所有電子書。每本書都以精美的卡片呈現,包含封面、標題、頁數等資訊。您可以輕鬆上傳新書、管理現有書籍,或透過 QR Code 和連結分享給他人。系統支援 PDF 檔案和圖片檔案,並會自動處理轉換工作。

### 電子書閱讀器
閱讀器提供專業級的閱讀體驗,配備完整的控制工具欄。您可以使用縮放功能調整頁面大小,使用書籤標記重要內容,透過縮圖面板快速跳轉到任何頁面,或進入全螢幕模式享受沉浸式閱讀。頁面切換動畫流暢自然,鍵盤快捷鍵讓操作更加便捷。

### 分享功能
每本電子書都會自動生成專屬的分享連結和 QR Code。您可以透過這些方式將電子書分享給任何人,接收者無需註冊即可立即開始閱讀。這讓知識的分享變得簡單而直接。

## 🚀 技術架構

### 前端技術
前端採用 React 18 搭配 TypeScript,確保代碼的類型安全和可維護性。Vite 作為建置工具提供極快的開發體驗,Tailwind CSS 配合自定義配置實現精美的視覺效果。Framer Motion 為所有互動添加流暢的動畫,提升整體質感。PDF.js 負責處理 PDF 檔案的轉換,確保高品質的圖片輸出。

### 後端服務
後端完全基於 Supabase 雲端平台,提供可靠的資料庫和檔案儲存服務。PostgreSQL 資料庫管理所有書籍和頁面資訊,Storage 服務存儲圖片檔案並提供 CDN 加速。這種架構確保了應用程式的可擴展性和高可用性。

### 部署方案
前端部署於 Vercel,享受全球 CDN 加速和自動化部署流程。Supabase 提供後端雲端服務,無需管理伺服器。這個組合讓您能專注於產品本身,而不必擔心基礎設施的維護。

## 📦 安裝與設置

### 環境需求
開始之前,請確保您的開發環境已安裝 Node.js 18 或更高版本,以及 npm 或 yarn 套件管理工具。您還需要註冊一個 Supabase 帳號來使用後端服務。

### 安裝步驟

首先,克隆專案到本地環境。在終端機中執行必要的套件安裝命令,這將下載所有相依套件。接著,您需要在 Supabase 建立新專案並設置資料庫結構。請參考 SUPABASE_SETUP.md 文件中的詳細指示來完成這個步驟。

設置完成後,創建 .env 檔案並填入您的 Supabase 專案資訊。最後,執行開發伺服器命令,即可在瀏覽器中訪問應用程式。

### 資料庫設置

Supabase 資料庫需要兩個主要資料表。books 資料表儲存電子書的基本資訊,包括標題、描述、總頁數和封面連結。book_pages 資料表儲存每一頁的圖片連結和頁碼。此外,您還需要建立名為 ebook-pages 的 Storage Bucket 來存放圖片檔案,並設置適當的存取權限。

詳細的 SQL 語句和權限設置請參考 SUPABASE_SETUP.md 文件。

## 🎨 設計特點

### 配色系統
平台採用精心設計的配色系統,以藍色作為主色調,紫色作為強調色。這個配色方案既專業又現代,適合各種使用情境。所有顏色都經過仔細調整,確保足夠的對比度和可讀性。

### 動畫效果
我們為所有互動添加了精心設計的動畫效果。頁面切換採用 3D 翻轉動畫,元素出現使用淡入和滑動效果,按鈕互動配備縮放和彈跳反饋。這些動畫不僅美觀,更重要的是提供了清晰的視覺反饋,讓用戶始終知道系統的狀態。

### 響應式設計
平台在各種裝置上都能完美運作。桌面版提供完整的功能和最佳的視覺效果,平板版優化了觸控操作和畫面佈局,手機版則提供簡化但同樣強大的功能。無論使用哪種裝置,用戶都能獲得流暢的體驗。

## 📱 使用指南

### 上傳電子書

點擊管理後台右上角的上傳按鈕,即可開啟上傳面板。您可以直接拖放檔案到指定區域,或點擊選擇檔案。系統支援 PDF 檔案和多種圖片格式。對於 PDF 檔案,系統會自動將每一頁轉換為高品質的 JPEG 圖片。上傳過程中,您可以看到即時的進度提示,完成後書籍會立即出現在您的書庫中。

### 閱讀電子書

點擊任何書籍卡片上的閱讀按鈕即可進入閱讀器。閱讀器提供多種控制選項,您可以使用畫面上的按鈕或鍵盤快捷鍵進行操作。左右方向鍵用於翻頁,B 鍵切換書籤,T 鍵開關縮圖面板。頁面之間的切換採用流暢的動畫效果,讓閱讀體驗更加自然。

### 分享電子書

每本書都可以透過分享功能分享給他人。點擊分享按鈕後,系統會顯示專屬的 QR Code 和連結。您可以複製連結直接傳送,或讓對方掃描 QR Code。接收者無需註冊或登入,點擊連結即可立即開始閱讀。

## 🔧 開發與部署

### 本地開發

開發模式下,Vite 提供熱模組替換功能,讓您能即時看到代碼更改的效果。TypeScript 會在開發過程中提供類型檢查,幫助您及早發現潛在問題。所有的環境變數都從 .env 檔案讀取,確保敏感資訊不會洩露。

### 生產建置

準備部署時,執行建置命令會生成優化後的生產版本。Vite 會自動進行代碼分割、壓縮和優化,確保最佳的載入效能。建置完成後的檔案會輸出到 dist 目錄,可以直接部署到任何靜態檔案託管服務。

### Vercel 部署

Vercel 提供最簡單的部署方式。在 Vercel 儀表板中匯入 GitHub repository,設置必要的環境變數,即可完成部署。之後,每次推送到主分支都會自動觸發新的部署,讓您能專注於開發而不必擔心部署流程。

## 🔒 安全性

平台採用多層安全措施保護用戶資料和系統安全。所有敏感資訊都透過環境變數管理,從不直接寫入代碼。Supabase 的 Row Level Security 政策確保資料存取權限得到妥善控制。所有檔案上傳都經過驗證,防止惡意檔案進入系統。通訊過程採用 HTTPS 加密,保護資料傳輸安全。
